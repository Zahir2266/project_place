# Event Management System API

Тестовое задание на позицию Python/Django разработчика.

## Стек технологий
- **Python 3.11**, **Django 4.2**, **DRF**
- **PostgreSQL** (Основная БД)
- **Redis + Celery + Celery Beat** (Фоновые задачи и парсинг погоды)
- **Docker + Docker Compose** (Контейнеризация)
- **Swagger (drf-spectacular)** (Документация API)

## Основной функционал
- [x] CRUD Мест и Мероприятий с разграничением прав.
- [x] Автоматическая генерация превью изображений (Pillow).
- [x] Фильтрация, поиск и пагинация (django-filter).
- [x] Асинхронная отправка Email и обновление погоды (Open-Meteo API).
- [x] Импорт/Экспорт данных в XLSX (openpyxl).
- [x] Автопубликация мероприятий по расписанию.
